DESCRIERE APLICATIE


Platforma "Biblioteca" - contine carti care pot fi imprumutate de useri, fiecare carte imprumutata aparand pe profilul userului resptectiv



FUNCTIONALITATE:


Etapa de logare/inregistrare:

In cazul in care utilizatorul are deja un cont, acesta se poate loga direct prin:
- Username
- Password

Daca utilizatorul este nou pe aplicatie, acesta isi poate face cont din Sign-UP prin:
- Username
- Name
- Surname
- Email
- Password

Functii user / administrator:

User:
- Se poate loga pe aplicatie
- Poate imprumuta carti disponibile
- Se poare deloga

Administrator
- Aceleasi funtii ca si userul
- Poare adauga carti in biblioteca
- Poate vizualiza profilurile user-ilor si cartile imprumutate de acestia
